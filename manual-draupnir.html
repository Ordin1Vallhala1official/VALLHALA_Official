STEP 1VPS（仮想デスクトップ）の導入 ※強く推奨

EAは24時間相場を監視して自動売買を行うため、ご自身のパソコンではなくVPS（仮想サーバー）での稼働を強く推奨いたします。（ご自宅のPCの電源が落ちたり、スリープ状態になるとEAも停止してしまうためです）。

☆PCをお持ちでない方でもVPSを使用することで、スマホのみで稼働ができます！

【代表的なおすすめVPSサービス】

  • ABLENET（エイブルネット）：FX自動売買の定番。通信が安定しています。
  • ConoHa for Windows Server：初期設定が簡単で初心者向けです。
  • お名前.com デスクトップクラウド：FX専用プランがあり安心です。
  • Hyonix(ハイオニクス)：海外のサービスですが安価で十分な容量が使用できます。

VPSを契約したら、提供元のマニュアルに従って「リモートデスクトップ接続」を行い、VPSのWindows画面を開いてください。以降の作業は、すべてVPSの画面上で行います。

STEP 2MT4のインストールとログイン

MT4のダウンロード

開設していただいた証券会社のホームページから「MT4（MetaTrader 4）」のPC用インストールファイルをダウンロードし、VPS上にインストールしてください。

口座へのログイン

MT4を起動すると「取引口座のログイン情報」を求められます。証券会社から送られてきた以下の情報を入力してログインします。

  • ログインID （口座番号）
  • パスワード
  • サーバー名 （※一文字でも違うとログインできませんのでご注意ください）

右下の接続状況に「〇〇/〇〇 kb」と数値が表示されればログイン成功です。（「無効な口座」等の表示は失敗しています）

STEP 3EAファイルとSetファイル（設定ファイル）の導入

お渡ししたダウンロードリンクから、EA本体（.ex4）と設定ファイル（.set）をVPS上に保存してください。

MT4の左上メニューから 「ファイル」 ＞ 「データフォルダを開く」 をクリックします。

EA本体の配置

開いたフォルダの中から 「MQL4」 ＞ 「Experts」 フォルダを開き、その中にダウンロードしたEAファイル（DRAUPNIR V 配布用.ex4）をコピー＆ペーストします。

Setファイルの配置

一つ上の階層（MQL4フォルダ）に戻り、今度は 「Presets」 フォルダを開きます。その中に設定ファイル（DRAUPNIR V 推奨セット.set）をコピー＆ペーストします。

MT4に戻り、左側の「ナビゲーター」パネル内にある「エキスパートアドバイザ」の上で右クリックし、「更新」を押します。リストに『DRAUPNIR V 配布用』が表示されれば導入成功です。

STEP 4【重要】MT4のEA稼働設定とAPI認証許可

DRAUPNIRは高度な認証システム（VALLHALA Web API）を搭載しているため以下の設定が必須となります。

MT4上部のメニューから 「ツール」 ＞ 「オプション」 を開きます。「エキスパートアドバイザ」タブを開き、以下の2箇所にチェックを入れます。

  • ☑ 自動売買を許可する
  • ☑ WebRequestを許可するURLリスト

「WebRequestを許可するURLリスト」のすぐ下にある「+（新しいURLを追加）」をダブルクリックし、以下のURLをコピー＆ペーストして入力します。

認証システム用： https://script.googleusercontent.com/

入力できたら「OK」ボタンを押して画面を閉じます。また、MT4の上部ツールバーにある「自動売買」ボタンが緑色（ON）になっていることを確認してください。赤色（OFF）の場合はクリックして緑色にします。

STEP 5チャートへのEA適用と稼働開始

チャートの準備

指定の通貨ペア（☆推奨通貨ペア: USDJPY, GBPJPY, GOLD）のチャートを開きます。チャート上部の時間足ボタンから、必ず 「M5（5分足）」 を選択してください。（DRAUPNIRは5分足専用に設計されています）

EAの適用

左側のナビゲーターパネルから『DRAUPNIR V 配布用』を、開いたチャート上にドラッグ＆ドロップします。

設定（Setファイル）の読み込み

設定画面が開きます。「全般」タブを開き、「自動売買を許可する」にチェックが入っているか確認します。

👑 DRAUPNIR v2.70 ユーザーマニュアル

システムの詳細ロジックとパラメーターの完全解説です。

■ 1. はじめに：DRAUPNIR（ドラウプニル）とは？

本EAは、相場のボラティリティ（変動幅）に自動追従する次世代AIロジックと、複数のテクニカル根拠を点数化する「スコアリング・システム」を搭載したリスク管理特化型EAです。

巷に溢れる「逆行したらただナンピンするだけのEA」とは異なり、レンジ相場を自動で回避し、強いトレンドが発生した時のみ「ピラミッティング（増し玉）」による追撃で利益の最大化を追求します。あなたの口座資金を徹底的に守り抜きながら、相場の波を乗りこなすスマートな資産運用を実現します。

※本システムは【VALLHALA Web API 認証システム】により保護されています。ご登録いただいた口座でのみ稼働する限定システムとなります。

■ 2. DRAUPNIR 5つの最強システム（特徴と強み）

  • レンジ相場回避 ＆ トレンド爆益ブースト機能
    相場が方向感のない「レンジ（揉み合い）」の時は、無駄なエントリーを自動でストップして損失を防ぎます。逆に、強いトレンドが発生したことを検知すると、一時的にエントリー基準を緩和し、初期ロットを増幅（Lot Boost）させて一気に利益を刈り取ります。
  • スコアリング・エントリーシステム
    「すべての条件が揃うまで待つ」のではなく、フィボナッチ到達、ローソク足の反発形状（プライスアクション）、オシレーターの状態などを「配点化」。合格ラインを超えたポイントでのみ高精度な狙撃を行います。
  • 鉄壁の守り：スマート・ナンピン
    含み損を抱えた場合でも、単なる値幅でナンピンはしません。「〇分以上経過」かつ「ローソク足の明確な反発サイン」が出た瞬間にのみ追加ポジションを持つため、急落・急騰相場での口座破綻リスクを極限まで抑えています。
  • 最大の攻め：ピラミッティング（順行追撃）
    予想通りに利益が乗っている強いトレンド相場では、利益を担保に「増し玉（ピラミッティング）」を行い、1回のトレンドで得られる利益を何倍にも膨らませます。
  • 完全自動の資金管理＆パーセンテージ決済
    複利機能をONにすれば、口座資金に合わせてロットを自動計算。さらに、複数ポジションを持った際は「現在の口座資金の〇%」という割合でトレイリング決済を行うため、ほったらかしでも常に最適な利益を確定し続けます。スワップポイント負けを防ぐ「最大保有日数決済」も備えた万全の設計です。

■ 3. パラメーター詳細設定ガイド

ご自身の運用スタイルや資金量に合わせて設定を変更できます。
※（★）マークの項目は、特にご確認いただきたい重要パラメーターです。

⚙️ メインロジック設定

相場環境の認識と、エントリーの土台となる設定です。

TradeDirection 取引方向の制限です。「0=両方稼働」「1=買いのみ」「2=売りのみ」。相場の大局に合わせて片側だけ稼働させたい場合に変更します。
HigherTF 上位足（大局のトレンド）を判断する時間足です。デフォルトの240は「4時間足」を意味します。
HTF_FastMA / HTF_SlowMA 上位足のトレンドを測るための移動平均線（短期・長期）の設定値です。
LTF_MAPeriod 現在のチャート（5分足）の移動平均線の設定値です。
FiboLookback 過去何本分のローソク足から最高値・最安値を割り出し、フィボナッチを引くかを決めます。
TargetFiboMode 狙う押し目・戻り目の深さ（38.2%、50.0%、61.8%）を7つのパターンから選択します。
FiboLevel1 / 2 / 3 フィボナッチの各ラインの数値です（基本的に変更不要です）。

🎯 【スコアリング】エントリー基準設定

DRAUPNIRの最大の特徴である、複数の根拠を点数化してエントリーする機能です。

EntryThreshold ★ エントリーを実行する合格ライン（点数）です。この数値を下げると手数が激増し、上げるとエントリーが厳選されます。
Score_Fibo / PA / RSI / BB 各条件（フィボナッチ到達、プライスアクション、RSI、ボリンジャーバンド）を満たした時に加算される配点です。

🔥 トレンド爆益ブースト設定

強いトレンドが発生した際に、利益を効率的に獲得するためのリミッター解除機能です。

UseDynamicThreshold トレンド発生時にエントリー基準を緩和（点数を下げる）するかどうか。
RelaxADXLevel この数値以上にトレンドが強くなった時（ADX値）に基準を緩和します。
RelaxedThreshold 緩和時の新しいエントリー合格ラインです。
UseTrendLotBoost トレンド時に初期ロットを大きくするかどうか。
BoostADXLevel / LotBoostMultiplier ADXがこの数値を超えたら、ロットを「何倍」に増幅させるかの設定です。

🛡️ レンジ相場回避フィルター

方向感のない相場（レンジ）での無駄な負けを減らすための盾です。

UseRangeFilter ★ レンジ回避機能を使用するかどうか。
ADX_Period トレンドの強さを測るインジケーター（ADX）の計算期間です。
ADX_MinLevel 最低限必要なトレンドの強さです。これ未満の時はエントリーをお休みします。
MA_MinDist_ATR 上位足の短期MAと長期MAが近すぎる（絡み合っている＝レンジ）時にエントリーを回避します。

📏 ATR（ボラティリティ自動追従）設定

ATR_Period ボラティリティを計算する期間です。
ToleranceATR_Mul ラインでの反発を判定する際の「許容誤差（どれくらいピッタリタッチを求めるか）」です。

📊 オシレーター・レジサポ設定

UseRSI / UseBB エントリー条件としてRSIやボリンジャーバンドの反発を含めるかどうかのON/OFFです。
RSI_BuyLevel / RSI_SellLevel RSIの売られすぎ・買われすぎの基準値です。
UseSR_Confluence 過去のレジスタンス・サポートライン（水平線）との重なりを根拠に含めるかどうかのON/OFFです。

💰 エントリー・ロット管理設定

InitialLot 複利運用をOFFにした場合の固定ロット数です。
MagicNumber EAを識別するためのマジックナンバーです。他のEAと同時に動かす場合は被らないように変更してください。
UseCompound ★ 口座資金に合わせてロットを自動計算する「複利運用」のON/OFFです。
RiskPercent ★ 複利運用時、口座資金の何％のリスクを取るかの設定です（0.04〜0.08推奨）。
UseCocomo 負けた次のトレードでロットを上げるココモ法のON/OFFです（通常はfalse推奨）。

⚔️ スマートナンピン設定(逆行時の守り)

UseNanpin ナンピン機能のON/OFFです。
Nanpin_ATR_Mul ナンピンを開始する距離です。相場が荒れている時は自動で幅が広がります。
NanpinMultiplier ナンピン時のロット増加倍率です。
MaxNanpin 最大何回までナンピンを許可するかの設定です。
UseSmartNanpin 指定距離まで落ちてもすぐには買わず、「反発のサイン（ローソク足の形）」が出るまで待つ賢いナンピンのON/OFFです。
NanpinMinMinutes 次のナンピンを行うまでの最低待機時間です。一気に撃ち尽くすのを防ぎます。

🚀 ピラミッティング設定(順行追撃)

UsePyramiding ★ 利益が乗っている方向に「増し玉（追撃）」を行う機能のON/OFFです。
Pyramid_ATR_Mul 追撃を行う利益幅です。
PyramidMultiplier 追撃時のロット倍率です。
MaxPyramid 最大何回まで追撃を許可するかの設定です。

🧺 動的バスケット決済設定 (超重要)

UseBasketClose トータル利益での一括決済（バスケット決済）のON/OFFです。
BasketTargetPct 口座資金の何％の利益が出たら一括決済するかの目標値です（例: 1.0なら資金の1%で利確）。
UseBasketTrailing 一括決済にトレイリングストップ（利益を伸ばす機能）を使うかどうかの設定です。
UseFailsafeSL ★ / FailsafeLossPercent ★ 口座を守る【最終防衛ライン】です。口座資金の指定％の含み損を抱えたら、全決済して強制的に逃げます。

⏳ スワップ対策(タイムリミット)

UseMaxHoldDays ★ / MaxHoldDays ★ ポジションを長期間保有し、マイナススワップ（日々の利息支払い）で資金が削られるのを防ぐため、指定した日数が経過したら強制決済します。

🔄 トレンド転換・フィルター設定

UseTrendClose 上位足のトレンドが「買い」から「売り」に変わった等、大局が逆転した際に、傷が浅いうちに損切りして逃げる機能です。
UseLTFTrendConf 下位足のMAがクロスしたことを確認してからエントリーする安全機能です。
UsePanicFilter フラッシュクラッシュのような異常な暴落・暴騰時に、エントリーを一時停止して身を守ります。
UseTimeFilter / UseNewsFilter 指定した時間帯や、特定の時間（指標発表時など）に取引を停止する機能です。

📱 通知設定

UsePushNotify スマホのMT4アプリへプッシュ通知を送ります。
UseDiscord / DiscordWebHook Discordの指定チャンネルへ取引通知を送るための設定です。

■ 4. 初心者向け：推奨稼働ステップ

  • まずは同梱のセットファイルの設定（デフォルト）で稼働開始
    本EAはデフォルト状態で「攻守のバランス」が最も最適化されるよう設計されています。まずは同梱のセットファイルを読み込んで稼働させてみてください。
  • 【安全重視の方】
    UseFailsafeSL を true に変更し、万が一の相場急変時に資金の大部分を守る設定（デフォルトは資金の10%でカット）にすることをお勧めします。
  • 【トレンド特化の方】
    もしご自身で「今は完全な上昇/下降トレンドだ」と判断できる場合は、TradeDirection を DIR_LONG_ONLY /SHORT_ONLY （買いのみ/売りのみ）に設定し、逆張りのリスクをゼロにして稼働させる使い方も非常に強力です。
